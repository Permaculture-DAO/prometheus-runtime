# Evidence batches

Status: `S6 draft`

This directory contains synthetic evidence-reference packages only.

The current runtime does not create MRV-authoritative evidence, does not perform Holochain commits, and does not admit production use.

Allowed:

- deterministic synthetic batch packages;
- event-reference hashes;
- simulated Holochain reference hashes.

Not allowed:

- real field evidence;
- real custody claims;
- Holochain entry commits;
- certification or MRV authority.
