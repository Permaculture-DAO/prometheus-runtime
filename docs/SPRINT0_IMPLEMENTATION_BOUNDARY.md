# Sprint 0 implementation boundary

This repository participates in the Prometheus bounded pilot deployment programme.

Sprint 0 establishes repository truth, source boundaries, access controls, review workflows and machine-readable execution evidence. It does not establish production readiness, scientific validation, legal admission, market admission, ecological value, token rights or investor rights.

## Controlling hierarchy

1. formally adopted canon;
2. controlling machine-readable schemas;
3. repository-specific contracts;
4. implementation and tests;
5. runtime outputs and UI.

Lower layers must not silently redefine higher layers.
